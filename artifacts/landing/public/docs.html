Lifegram Live Chat Widget — Setup Guide

@lifegrambot Documentation v1.0

What is Lifegram Widget?

Lifegram Widget is an embeddable live chat bubble — like Zendesk or Intercom — that you add to your website with a single line of code. Visitors start real-time conversations with you, and you respond from the Lifegram Mini App on Telegram.

💬 Real-time Chat

Visitors get instant replies via polling. No page refresh needed.

🎨 Custom Branding

Choose colors, icons, greeting, position, and logo text.

📱 Mobile Ready

Full-screen on mobile, floating bubble on desktop.

💾 Persistent Sessions

Chat history saved in localStorage with 7-day auto-expiry.

1 Create a Widget

Open the Lifegram Mini App in Telegram, go to the Setup tab, and tap Create Widget. Configure your settings:

SettingDescription
Site NameShown in the chat header (e.g. "My Store")
ColorBrand color for the bubble, header, and buttons
GreetingWelcome message shown to visitors
PositionLeft or right side of the screen
Bubble IconChat bubble, question mark, headset, or wave
Logo Text2-letter initials in the chat header circle

2 Copy the Embed Code

After creating a widget, tap Embed Code to reveal the snippet:

<script
  src="https://mini.susagar.sbs/api/w/embed.js?key=YOUR_KEY"
  data-key="YOUR_KEY"
  async>
</script>

Replace YOUR_KEY with your widget key from the Setup page.

3 Paste on Your Website

Add the embed code before the closing </body> tag on every page where you want the chat widget:

<!DOCTYPE html>
<html>
<head>...</head>
<body>
  <!-- Your website content -->

  <!-- Lifegram Widget -->
  <script src="https://mini.susagar.sbs/api/w/embed.js?key=YOUR_KEY"
          data-key="YOUR_KEY" async></script>
</body>
</html>

Works with any platform: WordPress, Shopify, Wix, Squarespace, static HTML, React, Next.js, and more.

4 Respond to Messages

When a visitor sends a message, it appears in your Widget Inbox tab inside the Lifegram Mini App. Reply in real-time — visitors see your responses within seconds.
